Jerman

Dalam arti Pasal 4 angka 7 General Data Protection Regulation (GDPR), operator bertindak sebagai Pengendali Data (Data Controller).

Pemrosesan data pribadi dilakukan sesuai dengan:

  • General Data Protection Regulation
  • Bundesdatenschutzgesetz

Route2DE merupakan proyek independen yang berbasis di Jerman.

2. Tujuan Pemrosesan Data

Route2DE adalah platform edukatif dan informatif yang memberikan panduan mengenai jalur legal menuju Jerman.

Data pribadi diproses secara terbatas dan hanya untuk:

  • Menyediakan akun pengguna
  • Mengaktifkan fungsi inti aplikasi
  • Menjaga stabilitas teknis dan keamanan sistem
  • Mencegah penyalahgunaan atau akses tidak sah

Route2DE tidak menggunakan data untuk tujuan pemasaran atau periklanan.

3. Kategori Data yang Dikumpulkan

A. Data Akun

  • Alamat email
  • Kredensial autentikasi terenkripsi
  • Tanggal pembuatan akun

B. Data Teknis

  • Jenis perangkat
  • Versi aplikasi
  • Log crash (jika diaktifkan)

C. Data Penggunaan (Opsional)

  • Data interaksi anonim untuk peningkatan performa aplikasi (jika analitik diaktifkan)

Route2DE tidak mengumpulkan data sensitif sebagaimana dimaksud dalam Pasal 9 GDPR (misalnya data kesehatan, agama, atau biometrik).

4. Dasar Hukum Pemrosesan (Pasal 6 GDPR)

Pemrosesan data dilakukan berdasarkan:

  • Pasal 6(1)(b) GDPR - Pelaksanaan kontrak (fungsi akun dan aplikasi)
  • Pasal 6(1)(f) GDPR - Kepentingan sah (keamanan dan stabilitas sistem)
  • Pasal 6(1)(a) GDPR - Persetujuan pengguna (jika fitur analitik opsional diaktifkan)

Route2DE tidak melakukan pengambilan keputusan otomatis atau profiling sebagaimana dimaksud dalam Pasal 22 GDPR.

5. Penyimpanan dan Pemrosesan Data (Firebase)

Route2DE menggunakan layanan Firebase yang disediakan oleh Google.

Google bertindak sebagai Pemroses Data (Data Processor) sesuai Pasal 28 GDPR dan memproses data atas nama Pengendali Data.

Perjanjian pemrosesan data dan Standard Contractual Clauses diterapkan sesuai ketentuan hukum Uni Eropa jika diperlukan.

6. Penyimpanan dan Retensi Data

  • Data akun disimpan selama akun aktif
  • Data teknis disimpan sementara untuk tujuan keamanan
  • Data akan dihapus secara permanen setelah akun dihapus

Route2DE tidak menyimpan data untuk tujuan pemasaran jangka panjang.

7. Hak Pengguna

Sesuai GDPR, pengguna memiliki hak untuk:

  • Mengakses data pribadi (Pasal 15)
  • Memperbaiki data (Pasal 16)
  • Menghapus data (Pasal 17)
  • Membatasi pemrosesan (Pasal 18)
  • Memindahkan data (Pasal 20)
  • Mengajukan keluhan kepada otoritas pengawas perlindungan data di Jerman

8. Penghapusan Data (Self-Service)

Pengguna dapat menghapus akun langsung melalui aplikasi Route2DE kapan saja.

Setelah penghapusan:

  • Data autentikasi akan dihapus permanen
  • Data terkait di database akan dihapus
  • Tidak ada pemrosesan lanjutan
  • Penghapusan bersifat permanen dan tidak dapat dibatalkan.

Ketentuan ini memenuhi hak atas penghapusan (Right to Erasure) sesuai Pasal 17 GDPR tanpa perlu menghubungi dukungan pelanggan.

9. Tidak Ada Penjualan Data

Route2DE tidak menjual, menyewakan, memperdagangkan, atau mengeksploitasi data pribadi untuk tujuan komersial.

Data pribadi tidak dibagikan kepada:

  • Platform periklanan
  • Jaringan pemasaran
  • Broker data
  • Pihak ketiga untuk iklan bertarget

Route2DE tidak menggunakan data pribadi untuk periklanan berbasis perilaku (behavioral advertising).

10. Perubahan Kebijakan

Kebijakan Privasi ini dapat diperbarui sewaktu-waktu untuk menyesuaikan perubahan hukum atau teknis.

Tanggal pembaruan terakhir akan dicantumkan di bagian atas dokumen ini.
